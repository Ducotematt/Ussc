If Unexpected Shoutdown Happned , Then It may require recover Wallet. To do recover wallet you need admin access in your Workstation. 

# Mac Recover Wallet Proccess :: 
## 1st : go to your wallet application directory from CLI interface
`cd /path/to/ussc-qt`
## 2nd : open wallet using admin previleges with recover argument
`sudo ./ussc-qt -salvagewallet`
